# Subscription Checkout Page

A responsive subscription checkout page built with HTML and CSS.

## Demo
https://krunxlbhoir.github.io/subscription-checkout/

## How to run locally
Open `index.html` in a web browser.

## Features
- Personal information and payment details section
- Plan selection section
- Responsive layout for desktop and mobile devices
- Clean and user-friendly UI

## Tech Stack
- HTML
- CSS (Grid, Flexbox, responsive design)

## Notes
- Layout created using CSS Grid for main structure
- Flexbox used for form rows and plan selection
- Responsive design implemented using media queries
